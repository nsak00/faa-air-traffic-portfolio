# FAA Air Traffic Control Network Analysis

This project features a comprehensive Social Network Analysis (SNA) of the US Federal Aviation Administration (FAA) Air Traffic Control Network, based on the KONECT dataset `maayan-faa`.

## Project Contents
- **Analytical Report:** [View the interactive web report](https://nsak00.github.io/faa-air-traffic-portfolio/)
- **Methodology:** Complete analysis using Python (NetworkX, Pandas) and Gephi.
- **Key Visualizations:** Includes degree distributions, community detection (Louvain), and cascading delay simulations (ICM).

## Quick Access
- Dataset Source: [KONECT - maayan-faa](http://konect.cc/networks/maayan-faa/)
